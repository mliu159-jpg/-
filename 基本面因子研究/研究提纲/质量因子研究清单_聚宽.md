# 质量因子研究清单（聚宽平台）

> 五大类基本面因子之四。质量因子衡量"企业的财务是否健康、利润是否真实、资产是否扎实"。它不直接预测高收益，更多是**风险防御**——帮你在价值股里剔除"价值陷阱"，避开财务造假和资金链断裂。质量因子与估值因子是天然搭档。

---

## 一、因子总览

| 因子族 | 核心因子 | 聚宽来源 | 预期方向 | 说明 |
|---|---|---|---|---|
| 杠杆 / 偿债 | 资产负债率、有息负债率、利息保障倍数 | `balance` / `income` 自算 | 杠杆越低越好 | 衡量财务风险 |
| 流动性 | 流动比率、速动比率 | `balance` 自算 | 越高越好 | 短期偿债能力 |
| 营运效率 | 总资产 / 应收 / 存货周转率 | `income` / `balance` 自算 | 越高越好 | 资产使用效率 |
| 盈利质量 | 应计利润、现金含量 | `cash_flow` / `income` 自算 | 应计越低越好 | 识别利润注水 |
| 盈利稳定性 | 多期 ROE / 利润的波动率 | 自算 | 波动越低越好 | 抗周期能力 |
| 资产质量 | 商誉占比、应收账款占比 | `balance` 自算 | 占比越低越好 | 资产"水分" |
| 综合质量 | Piotroski F-Score | 多表组合 | 分数越高越好 | 9 项二元打分 |

---

## 二、聚宽数据获取基础

质量因子大多没有现成字段，需要从三大报表自己算，因此**口径自己掌控、过程透明**是它的特点。涉及的表：

- `balance`（资产负债表）—— 时点值
- `income`（利润表）—— 单季度值
- `cash_flow`（现金流量表）—— 单季度值

### 关键字段

| 字段 | 含义 |
|---|---|
| `balance.total_assets` / `balance.total_liability` | 资产总计 / 负债合计 |
| `balance.total_current_assets` / `balance.total_current_liability` | 流动资产 / 流动负债合计 |
| `balance.inventories` / `balance.account_receivable` | 存货 / 应收账款 |
| `balance.good_will` | 商誉 |
| `balance.shortterm_loan` / `longterm_loan` / `bonds_payable` | 短期借款 / 长期借款 / 应付债券 |
| `income.operating_revenue` / `income.net_profit` | 营业收入 / 净利润 |
| `income.operating_profit` / `income.financial_expense` | 营业利润 / 财务费用 |
| `cash_flow.net_operate_cash_flow` | 经营活动产生的现金流量净额 |

资产负债表是时点数，可直接用；利润表和现金流量表是单季度值，做比率（如周转率、现金含量）时要注意分子分母口径一致——单季度营收配单季度，或都用 TTM。

---

## 三、因子逐项说明

### 1. 杠杆 / 偿债

定义：资产负债率 = 总负债 / 总资产；有息负债率 = 有息负债 / 总资产；利息保障倍数 = EBIT / 利息费用。

杠杆因子衡量财务风险——负债越重，经营稍有波动就可能资金链紧张。在深度价值策略里它尤其重要：很多"看起来便宜"的股票便宜是因为高负债拖累。利息保障倍数衡量"利润够不够还利息"，倍数低于 1 是危险信号。方向：杠杆类越低越好（负向因子），利息保障倍数越高越好。

### 2. 流动性

定义：流动比率 = 流动资产 / 流动负债；速动比率 = (流动资产 − 存货) / 流动负债。

衡量短期偿债能力。速动比率把变现慢的存货剔除，更严格。一般流动比率 2 左右、速动比率 1 左右算健康，但行业差异大。方向：越高越好（但过高也可能是资产闲置，不必线性外推）。

### 3. 营运效率（周转率）

定义：总资产周转率 = 营业收入 / 总资产；应收账款周转率 = 营业收入 / 应收账款；存货周转率 = 营业成本 / 存货。

周转率衡量"资产转得快不快、钱收得回不回得来"。应收账款周转率突然下降，常是放宽信用冲收入、回款变差的前兆。存货周转率下降可能是产品滞销。方向：越高越好。注意周转率行业差异极大（重资产 vs 轻资产），必须行业中性化。

### 4. 盈利质量（应计利润）

定义：应计利润 accruals = (净利润 − 经营活动现金流) / 总资产；现金含量 = 经营活动现金流 / 净利润。

这是质量因子里**最有学术支撑**的一个——Sloan 的应计异象：应计利润高（利润里现金含量低）的公司，未来收益往往更差，因为这种利润可持续性弱、甚至是操纵出来的。低应计 = 高质量。方向：应计利润越低越好（负向因子），现金含量越高越好。

### 5. 盈利稳定性

定义：过去若干期 ROE 或净利润的标准差 / 变异系数。

利润忽高忽低的公司，要么是强周期行业，要么是经营不稳。稳定性因子识别"穿越周期的好生意"。它和盈利能力因子里的同名概念一致——可放在任一类，重点是别漏掉。方向：波动越低越好。

### 6. 资产质量

定义：商誉占比 = 商誉 / 总资产；应收账款占比 = 应收账款 / 总资产。

商誉是并购溢价形成的"虚资产"，占比过高意味着潜在的商誉减值地雷（A 股年报季的减值暴雷多与此有关）。应收账款占比过高说明大量收入挂账未收现。方向：占比越低越好（负向因子）。

### 7. 综合质量：Piotroski F-Score

定义：Piotroski 提出的 9 项二元打分（每项满足记 1 分），覆盖盈利性（ROA 为正、经营现金流为正、ROA 改善、现金流 > 净利润）、杠杆与流动性（负债率下降、流动比率上升、未增发新股）、运营效率（毛利率上升、资产周转率上升），满分 9 分。

F-Score 是把多个质量维度浓缩成一个分数的经典综合因子，**最初就是设计来在低估值股票里筛出真正健康的那批**，和深度价值策略契合度极高。方向：分数越高越好。

## 四、标准变换

质量因子里有不少**负向因子**（资产负债率、应计利润、商誉占比），构造打分时要先统一方向（乘 −1 或取倒数），让"数值大 = 质量好"。其余变换与前几类一致：行业中性化（周转率、杠杆尤其需要）、历史分位、多期均值与波动。质量因子常作为价值股的"过滤器"使用——先按估值选出便宜的，再用质量因子剔除尾部。

## 五、可直接运行的批量获取代码

```python
from jqfactor import winsorize_med, standardlize, neutralize
import pandas as pd
import numpy as np

def get_quality_factors(date, stock_list):
    """取指定日期的质量因子，已做标准预处理（已统一为正向：大=质量好）"""
    q = query(
        valuation.code,
        balance.total_assets,
        balance.total_liability,
        balance.total_current_assets,
        balance.total_current_liability,
        balance.inventories,
        balance.account_receivable,
        balance.good_will,
        balance.shortterm_loan,
        balance.longterm_loan,
        balance.bonds_payable,
        balance.non_current_liability_in_one_year,
        income.operating_revenue,
        income.operating_cost,
        income.net_profit,
        cash_flow.net_operate_cash_flow,
    ).filter(valuation.code.in_(stock_list))
    df = get_fundamentals(q, date=date).set_index('code')
    fillz = lambda s: s.fillna(0)

    f = pd.DataFrame(index=df.index)
    # 杠杆：取负，使"大=低杠杆=质量好"
    f['low_leverage'] = -(df['total_liability'] / df['total_assets'])
    # 流动性
    f['current_ratio'] = (df['total_current_assets']
                          / df['total_current_liability'])
    f['quick_ratio'] = ((df['total_current_assets'] - fillz(df['inventories']))
                        / df['total_current_liability'])
    # 营运效率
    f['asset_turnover'] = df['operating_revenue'] / df['total_assets']
    f['inv_turnover'] = df['operating_cost'] / df['inventories']
    f['ar_turnover'] = df['operating_revenue'] / df['account_receivable']
    # 盈利质量：应计取负，使"大=低应计=质量好"
    f['low_accruals'] = -((df['net_profit'] - df['net_operate_cash_flow'])
                          / df['total_assets'])
    # 资产质量：商誉占比取负
    f['low_goodwill'] = -(fillz(df['good_will']) / df['total_assets'])

    # 标准预处理（此时所有因子均为"越大越好"）
    for col in f.columns:
        s = f[col].replace([np.inf, -np.inf], np.nan)
        s = winsorize_med(s, scale=3, inclusive=True)
        s = standardlize(s, inf2nan=True)
        s = neutralize(s, how=['jq_l1', 'market_cap'], date=date)
        f[col] = s
    return f
```

F-Score 因为含"同比改善"项，需要多期数据，用 `get_history_fundamentals` 取相邻两期后逐项打分即可，逻辑直接但代码较长，可单独实现。

## 六、因子方向与检验提示

质量因子的特点是 **IC 不高但很稳、回撤小**——它的作用主要是降低组合的尾部风险，而不是冲收益。检验时除了看 IC，更要看它能否降低组合的最大回撤和波动。验证质量因子最有说服力的方式是：在一个纯估值策略上叠加质量过滤，看夏普比率和回撤是否改善。应计利润因子在 A 股有较稳定的负向预测力，值得重点测。

---

## 七、注意事项与坑

- **先统一方向**：资产负债率、应计利润、商誉占比是负向因子，合成打分前务必统一成"大=好"。
- **行业属性强**：金融、地产天然高杠杆，公用事业周转率天然低，杠杆和周转因子必须行业中性化，金融股常单独剔除。
- **分母为零 / 极小**：周转率、流动比率的分母（应收账款、存货、流动负债）可能为零或极小，会产生极端值，去极值前先处理 inf。
- **现金流季节性**：经营现金流单季度波动大（很多公司四季度集中回款），算现金含量建议用 TTM。
- **质量不替代估值**：再健康的公司买贵了也会亏，质量因子是过滤器不是选股主力，必须和估值配合。
