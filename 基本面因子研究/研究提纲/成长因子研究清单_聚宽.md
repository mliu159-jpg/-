# 成长因子研究清单（聚宽平台）

> 五大类基本面因子之二。成长因子衡量"企业基本面在变好还是变差"，是把时间维度引入选股的一类。注意成长与"深度价值"常处于对立面——本清单的价值在于让你把成长作为可对冲、可叠加的一类因子来检验，而不是默认它和价值同向。

---

## 一、因子总览

| 因子族 | 核心因子 | 聚宽来源 | 预期方向 | 说明 |
|---|---|---|---|---|
| 营收增长 | 营收同比、营业总收入同比 | `indicator.inc_revenue_year_on_year` | 越高越好 | 增长最干净的口径 |
| 利润增长 | 净利润同比、归母净利润同比、营业利润同比 | `indicator.inc_net_profit_year_on_year` 等 | 越高越好 | 易受一次性损益干扰 |
| 扣非利润增长 | 扣非净利润同比 | `indicator.adjusted_profit` 自算 | 越高越好 | 剔除非经常损益，更真实 |
| 每股指标增长 | EPS 同比增速 | `indicator.eps` 自算 | 越高越好 | 已考虑股本摊薄 |
| 复合增长率 | 营收 / 利润 3～5 年 CAGR | `get_history_fundamentals` | 越高越好 | 平滑单期波动 |
| 盈利能力改善 | ΔROE、ΔROA、Δ毛利率 | `indicator.roe` 自算 | 越高越好 | "质量改善"型成长 |
| 增长加速度 | 增速的环比变化（二阶） | `indicator.inc_*_annual` | 越高越好 | 增长拐点信号 |
| 增长稳定性 | 过去若干期增速的波动率 | 自算 | 波动越低越好 | 稳定增长优于忽高忽低 |

---

## 二、聚宽数据获取基础

### `indicator` 表关键字段（已验证）

| 字段 | 含义 |
|---|---|
| `indicator.inc_revenue_year_on_year` | 营业收入同比增长率(%) |
| `indicator.inc_total_revenue_year_on_year` | 营业总收入同比增长率(%) |
| `indicator.inc_net_profit_year_on_year` | 净利润同比增长率(%) |
| `indicator.inc_net_profit_to_shareholders_year_on_year` | 归母净利润同比增长率(%) |
| `indicator.inc_operation_profit_year_on_year` | 营业利润同比增长率(%) |
| `indicator.inc_revenue_annual` / `inc_net_profit_annual` | 对应的环比增长率(%) |
| `indicator.adjusted_profit` | 扣除非经常损益后的净利润（元） |
| `indicator.eps` | 每股收益（元） |

### 单季度口径

聚宽财报是**单季度值**，所以 `indicator` 里的同比增速默认是"本季度 vs 去年同季度"。如果你要的是 TTM（滚动 12 个月）增速或累计增速，得自己取多期数据滚动求和后再算。研究上建议明确固定一种口径——单季度同比反应快但噪声大，TTM 同比更平滑。

### 多期数据：`get_history_fundamentals`

CAGR、增长稳定性、增长加速度都需要多期数据。用 `get_history_fundamentals(securities, fields, watch_date=..., count=N, interval='4q')` 一次取多个报告期，`interval='4q'` 表示按年度同期取，`watch_date` 保证只取该日之前已公告的数据（防未来函数）。

## 三、因子逐项说明

### 1. 营收增长

定义：营业收入同比增速。来源 `indicator.inc_revenue_year_on_year`（或营业总收入口径 `inc_total_revenue_year_on_year`）。

营收是利润表最上游、最难操纵的科目，所以营收增长被认为是"最干净"的成长信号。利润可以靠费用调节、资产处置短期粉饰，营收不行。方向：越高越好。

### 2. 利润增长

定义：净利润 / 归母净利润 / 营业利润的同比增速。来源 `indicator.inc_net_profit_year_on_year`、`inc_net_profit_to_shareholders_year_on_year`、`inc_operation_profit_year_on_year`。

归母净利润口径剔除了少数股东损益，对选股更合适。营业利润口径剔除了营业外收支，更贴近主业。方向：越高越好，但要警惕由非经常损益堆出来的"假增长"。

### 3. 扣非利润增长

定义：扣除非经常性损益后净利润的同比增速。聚宽无现成的扣非增速字段，用 `indicator.adjusted_profit` 取本期与去年同期，自己算 `本期/去年同期 − 1`。

非经常损益（政府补贴、资产处置、公允价值变动）会让利润大起大落。扣非增速过滤掉这部分，是判断"主业是否真的在增长"的关键。建议把它和普通净利润增速做差，差值大说明增长含金量低。方向：越高越好。

### 4. 每股指标增长

定义：EPS 同比增速。用 `indicator.eps` 自算。

EPS 增速相比净利润增速多考虑了一层股本变化——靠增发做大利润但摊薄每股的公司，EPS 增速会暴露问题。方向：越高越好。

### 5. 复合增长率（CAGR）

定义：营收或利润在过去 3～5 年的几何平均增速，`CAGR = (期末值 / 期初值)^(1/年数) − 1`。

单期同比受基数效应和偶发因素影响大，CAGR 用多年数据平滑，更能反映"持续成长能力"。用 `get_history_fundamentals` 取多年年报数据计算。注意：期初或期末为负时几何平均没有意义，需单独剔除或改用其他口径。方向：越高越好。

### 6. 盈利能力改善（ΔROE 等）

定义：ROE、ROA、毛利率的同比变化量，如 `ΔROE = 本期ROE − 去年同期ROE`。

这是"质量改善"型成长——不是看规模扩大，而是看赚钱效率在提升。一个 ROE 从 8% 升到 12% 的公司，往往比一个营收翻倍但 ROE 不变的公司更有价值。ΔROE 在 A 股是较稳健的成长类因子。方向：越高越好。

### 7. 增长加速度（二阶）

定义：增速本身的变化，如"本季营收增速 − 上季营收增速"，或直接用环比增速 `inc_revenue_annual`。

捕捉的是增长拐点——市场往往对"增速由降转升"反应强烈。属于偏交易型的成长因子，换手较高。方向：加速度为正较好。

### 8. 增长稳定性

定义：过去若干期增速的标准差或变异系数。

同样是 20% 的年均增速，每年稳定 20% 的公司，质量高于"30%、5%、35%"忽高忽低的公司。稳定性因子是对增速因子的重要补充。方向：波动越低越好（注意它是负向因子）。

## 四、标准变换

成长因子同样可经标准变换扩展成族：原始增速、行业中性化排名、与估值结合（PEG = PE / 增速）、增速的分位数、增速的同比变化（加速度）。其中**与估值结合**对成长因子尤其重要——单纯买高增速容易追高，GARP（合理价格成长）思路通常更稳健。完整变换框架见《估值因子研究清单》第四节。

## 五、可直接运行的批量获取代码

```python
from jqfactor import winsorize_med, standardlize, neutralize
import pandas as pd
import numpy as np

def get_growth_factors(date, stock_list):
    """取指定日期的成长因子，已做标准预处理"""
    q = query(
        valuation.code,
        indicator.inc_revenue_year_on_year,                     # 营收同比
        indicator.inc_net_profit_year_on_year,                  # 净利润同比
        indicator.inc_net_profit_to_shareholders_year_on_year,  # 归母净利润同比
        indicator.inc_operation_profit_year_on_year,            # 营业利润同比
    ).filter(valuation.code.in_(stock_list))
    df = get_fundamentals(q, date=date).set_index('code')
    df = df.rename(columns={
        'inc_revenue_year_on_year': 'rev_growth',
        'inc_net_profit_year_on_year': 'np_growth',
        'inc_net_profit_to_shareholders_year_on_year': 'np_parent_growth',
        'inc_operation_profit_year_on_year': 'op_growth',
    })
    # 标准预处理
    for col in df.columns:
        s = df[col].replace([np.inf, -np.inf], np.nan)
        s = winsorize_med(s, scale=3, inclusive=True)
        s = standardlize(s, inf2nan=True)
        s = neutralize(s, how=['jq_l1', 'market_cap'], date=date)
        df[col] = s
    return df

# 多期因子示例：用 get_history_fundamentals 算扣非利润 3 年 CAGR
def get_adjusted_profit_cagr(date, stock_list, years=3):
    hist = get_history_fundamentals(
        stock_list,
        fields=[indicator.adjusted_profit],
        watch_date=date, count=years + 1, interval='4q')   # 取 years+1 个年度同期
    cagr = {}
    for code, g in hist.groupby('code'):
        v = g.sort_values('statDate')['adjusted_profit'].values
        if len(v) >= years + 1 and v[0] > 0 and v[-1] > 0:   # 首末为正才有意义
            cagr[code] = (v[-1] / v[0]) ** (1.0 / years) - 1
    return pd.Series(cagr, name='adj_profit_cagr')
```

## 六、因子方向与检验提示

成长因子的检验要特别关注**风格周期**：成长因子在牛市、流动性宽松期表现强，在熊市、价值占优期会跑输——这是因子特性。逐期看 IC 时，成长因子的 IC 序列波动通常比估值因子大，ICIR 偏低，所以**单用成长因子做策略风险较高，更适合与估值、质量因子组合**。换手率方面，增速类因子（尤其单季度同比、加速度）换手高，回测要把交易成本算进去。

---

## 七、注意事项与坑

- **基数效应**：去年同期利润极低（甚至亏损）会让今年同比增速虚高到几百上千个百分点，去极值时务必处理这种极端值。
- **不可持续性**：高增速本身有均值回归倾向，"高增长能否持续"才是关键，CAGR 和增长稳定性就是为此设计的。
- **一次性损益**：普通净利润增速含非经常损益，建议优先用扣非口径，或把两者之差作为"增长质量"因子。
- **单季度噪声**：单季度同比受去年同期偶发因素干扰大，必要时改用 TTM 增速。
- **防未来函数**：用 `get_history_fundamentals` 时必须传 `watch_date`，确保只用当时已公告的数据。
- **不要默认成长=好**：成长因子在 A 股长周期的超额收益并不稳定，把它当作一个待检验的中性因子，而非默认有效。
