# 盈利能力因子研究清单（聚宽平台）

> 五大类基本面因子之三。盈利能力因子衡量"企业用资产/资本/收入赚钱的效率"，是质量投资的核心，也是巴菲特式"好公司"思路的量化对应。它与估值因子结合（好公司 + 便宜价格）通常是 A 股最稳健的多因子组合之一。

---

## 一、因子总览

| 因子族 | 核心因子 | 聚宽来源 | 预期方向 | 说明 |
|---|---|---|---|---|
| 净资产回报 | ROE、扣非 ROE | `indicator.roe` / `indicator.inc_return` | 越高越好 | 最核心的盈利能力指标 |
| 总资产回报 | ROA | `indicator.roa` | 越高越好 | 不受杠杆影响 |
| 投入资本回报 | ROIC | 需自算 | 越高越好 | 衡量真实经营回报 |
| 利润率 | 毛利率、净利率、营业利润率 | `indicator.gross_profit_margin` 等 | 越高越好 | 反映定价权与成本控制 |
| 杜邦三因子 | 净利率 × 周转率 × 权益乘数 | 拆解 ROE | 见说明 | 识别 ROE 的驱动来源 |
| 盈利稳定性 | 多期 ROE 的均值 / 波动率 | 自算 | 均值高、波动低 | 稳定高回报是质量信号 |
| 盈利现金含量 | 经营现金流 / 净利润 | `cash_flow` / `income` 自算 | 越高越好 | 与质量因子衔接 |

## 二、聚宽数据获取基础

### `indicator` 表关键字段（已验证）

| 字段 | 含义 | 单位 |
|---|---|---|
| `indicator.roe` | 净资产收益率 ROE | % |
| `indicator.inc_return` | 净资产收益率（扣除非经常损益） | % |
| `indicator.roa` | 总资产净利率 ROA | % |
| `indicator.net_profit_margin` | 销售净利率 | % |
| `indicator.gross_profit_margin` | 销售毛利率 | % |
| `indicator.adjusted_profit` | 扣非净利润 | 元 |
| `indicator.operating_profit` | 经营活动净收益 | 元 |

### 口径提示

`indicator` 的 ROE/ROA 是基于**单季度**报表的，所以单季度 ROE 有明显季节性（四季度通常最高）。研究上建议用年报口径（取 `statDate` 为年末的记录），或用 TTM 净利润自己除以净资产算 TTM-ROE。利润率类指标的单季度波动相对小，但同样建议口径统一。

## 三、因子逐项说明

### 1. ROE（净资产收益率）

定义：ROE = 归母净利润 / 归母净资产。来源 `indicator.roe`；扣非口径用 `indicator.inc_return`。

ROE 是盈利能力的核心——它直接回答"股东每投入一元能赚回多少"。但 ROE 有个陷阱：高杠杆会放大 ROE（权益乘数大），所以高 ROE 不一定是好生意，可能只是借得多。这正是要做杜邦分解的原因。优先用**扣非 ROE**，剔除非经常损益后的回报才是真实经营能力。方向：越高越好。

### 2. ROA（总资产净利率）

定义：ROA = 净利润 / 总资产。来源 `indicator.roa`。

ROA 的分母是全部资产，不区分自有还是借来，因此**不受资本结构影响**，是比 ROE 更"纯粹"的经营效率指标。ROE 高但 ROA 平庸，基本可以断定回报是杠杆堆出来的。方向：越高越好。

### 3. ROIC（投入资本回报率）

定义：ROIC = EBIT × (1 − 税率) / 投入资本，投入资本 = 有息负债 + 股东权益。聚宽无现成字段，需自算。

ROIC 衡量"全部投入的资本（不管来自股东还是债主）创造的税后经营回报"，剔除了财务杠杆和非经营性损益的干扰，是最接近"生意本身好不好"的指标。ROIC 持续高于资本成本（WACC）的公司才是真正在创造价值。计算稍繁，但值得做。方向：越高越好。

### 4. 利润率（毛利率 / 净利率 / 营业利润率）

定义：毛利率 = (营收 − 营业成本) / 营收；净利率 = 净利润 / 营收；营业利润率 = 营业利润 / 营收。来源 `indicator.gross_profit_margin`、`indicator.net_profit_margin`，营业利润率可用 `indicator.operation_profit_to_total_revenue` 或自算。

毛利率反映**定价权和产品竞争力**，是护城河的直接体现；净利率在毛利基础上再反映费用控制和税负。三者行业差异极大（软件毛利率可达 80%+、零售可能个位数），所以**必须做行业中性化**，否则因子会退化成行业选择。方向：越高越好。

### 5. 杜邦三因子

定义：ROE = 净利率 × 总资产周转率 × 权益乘数。

杜邦分解把 ROE 拆成三个驱动力——盈利能力（净利率）、运营效率（周转率）、财务杠杆（权益乘数）。三个分项都可以单独作为因子检验。它的真正价值是**诊断**：同样 15% 的 ROE，靠高净利率（如品牌消费品）还是靠高杠杆（如银行地产），投资含义完全不同。建议把"高 ROE 且低权益乘数"作为一个组合条件。

### 6. 盈利稳定性

定义：过去若干年 ROE 的均值与标准差（或变异系数）。

一家 ROE 常年稳定在 18% 的公司，质量远高于 ROE 在 5%～30% 之间剧烈波动的公司——后者往往是强周期股。盈利稳定性因子（高均值 + 低波动）是识别"穿越周期的好生意"的关键。方向：均值越高越好、波动越低越好。

### 7. 盈利现金含量

定义：经营活动现金流净额 / 净利润。

利润是权责发生制下的会计数字，现金才是真金白银。这个比率持续显著低于 1（利润收不回现金）是盈利质量差、甚至利润注水的警报。它和质量因子里的"应计利润"是一体两面，详见《质量因子研究清单》。方向：越高越好（通常以 1 附近为健康）。

## 四、标准变换

盈利能力因子的常用变换：原始值、行业中性化（几乎必做）、历史分位（与自身历史比）、同比变化（ΔROE，已归入成长因子）、多期均值与波动（盈利稳定性）。与估值因子结合是重点——"高 ROE + 低估值"的双因子打分模型是 A 股久经检验的组合。完整变换框架见《估值因子研究清单》第四节。

## 五、可直接运行的批量获取代码

```python
from jqfactor import winsorize_med, standardlize, neutralize
import pandas as pd
import numpy as np

def get_profitability_factors(date, stock_list):
    """取指定日期的盈利能力因子，已做标准预处理"""
    # 1. 直接可取的指标
    q = query(
        valuation.code,
        indicator.roe,                  # 净资产收益率
        indicator.inc_return,           # 扣非 ROE
        indicator.roa,                  # 总资产净利率
        indicator.gross_profit_margin,  # 销售毛利率
        indicator.net_profit_margin,    # 销售净利率
        # 自算 ROIC 需要的原始项
        income.net_profit,
        income.total_profit,
        income.operating_profit,
        income.financial_expense,
        balance.total_owner_equities,
        balance.shortterm_loan,
        balance.longterm_loan,
        balance.bonds_payable,
        balance.non_current_liability_in_one_year,
    ).filter(valuation.code.in_(stock_list))
    df = get_fundamentals(q, date=date).set_index('code')

    # 2. 自算 ROIC（近似口径）
    tax_rate = (df['total_profit'] - df['net_profit']) / df['total_profit']
    tax_rate = tax_rate.clip(0, 0.4).fillna(0.25)                  # 防异常税率
    ebit = df['operating_profit'] + df['financial_expense'].fillna(0)
    interest_debt = (df[['shortterm_loan', 'longterm_loan',
                         'bonds_payable',
                         'non_current_liability_in_one_year']]
                     .fillna(0).sum(axis=1))
    invested_capital = interest_debt + df['total_owner_equities']
    roic = ebit * (1 - tax_rate) / invested_capital

    factors = df[['roe', 'inc_return', 'roa',
                  'gross_profit_margin', 'net_profit_margin']].copy()
    factors['roic'] = roic

    # 3. 标准预处理
    for col in factors.columns:
        s = factors[col].replace([np.inf, -np.inf], np.nan)
        s = winsorize_med(s, scale=3, inclusive=True)
        s = standardlize(s, inf2nan=True)
        s = neutralize(s, how=['jq_l1', 'market_cap'], date=date)
        factors[col] = s
    return factors
```

ROIC 用的是单季度报表，分子 EBIT 与利润是单季度值；做严肃研究时建议把利润项滚动成 TTM 再算，口径更稳。

## 六、因子方向与检验提示

盈利能力因子在 A 股通常表现稳健、换手低、单调性好，尤其是 ROE 和毛利率。检验时重点看：因子是否被行业中性化干净（未中性化的毛利率 IC 往往是行业差异的假象）；是否存在"盈利能力均值回归"——极高 ROE 难以维持，所以盈利*稳定性*往往比盈利*水平*更有预测力。盈利能力因子与估值因子相关性低，是优秀的互补组合。

---

## 七、注意事项与坑

- **ROE 的杠杆陷阱**：高 ROE 可能来自高负债，务必配合 ROA、权益乘数或资产负债率一起判断。
- **行业中性化必做**：利润率的行业天然差异巨大，不中性化因子几乎无效。
- **负净资产**：净资产为负时 ROE 失真（负净利润 / 负净资产会得到正值），需剔除。
- **季节性**：单季度 ROE 有季节性，统一用年报口径或 TTM。
- **均值回归**：盈利能力极值会回归，纯买"最高 ROE"长期未必占优，结合稳定性与估值更好。
- **金融股特殊**：银行、保险的盈利结构特殊，多因子研究中常单独剔除或单独建模。
