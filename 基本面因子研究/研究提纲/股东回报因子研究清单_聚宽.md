# 股东回报因子研究清单（聚宽平台）

> 五大类基本面因子之五。股东回报因子衡量"公司以现金形式回馈股东的力度与持续性"。它与深度价值高度契合——愿意持续真金白银分红的公司，往往同时具备稳定现金流、低估值和良好治理。这一类在 A 股近年随红利策略走强而愈发重要。

---

## 一、因子总览

| 因子族 | 核心因子 | 聚宽来源 | 预期方向 | 说明 |
|---|---|---|---|---|
| 股息率 | 近 12 个月每股分红 / 股价 | `finance.STK_XR_XD` 自算 | 越高越好 | 红利策略的核心 |
| 分红率 | 现金分红 / 归母净利润 | 分红表 + `income` | 见说明 | 派息意愿，非越高越好 |
| 分红连续性 | 连续分红年数、分红稳定性 | `finance.STK_XR_XD` 自算 | 越高越好 | 治理与现金流质量信号 |
| 股票回购 | 回购金额 / 市值 | 公告数据，可得性有限 | 越高越好 | A 股规模仍小 |
| 综合股东回报率 | (分红 + 回购) / 市值 | 组合 | 越高越好 | 广义股东收益率 |
| 股权稀释 | 总股本同比增长率 | `valuation.capitalization` | 越低越好 | 增发 / 送股摊薄，负向 |

---

## 二、聚宽数据获取基础

股东回报因子的核心数据是**分红数据**，不在三大报表里，需要查询专门的财务数据表。

### 分红数据：`finance.STK_XR_XD`

用 `finance.run_query(query(finance.STK_XR_XD).filter(...))` 查询除权除息表。该表记录每次分红的方案，关键信息包括股票代码、除权除息日、每股现金红利等。**字段名与单位以聚宽 API 文档为准**——尤其注意 A 股每股派息常以"每 10 股派 X 元"的形式登记，换算成每股时要除以 10。查询时按除息日筛选目标时间窗（如过去 12 个月）。

### 股本数据

`valuation.capitalization` 为总股本（单位：万股），`valuation.circulating_cap` 为流通股本。股权稀释因子用总股本的同比变化计算。

### 防未来函数

分红方案有"董事会预案 → 股东大会通过 → 实施"多个时点。回测中应以**除息日**或**实施公告日**为准来判断分红在某交易日是否"已知"，不能用预案日。

## 三、因子逐项说明

### 1. 股息率（Dividend Yield）

定义：股息率 = 近 12 个月每股现金分红合计 / 当前股价。

股息率是红利策略的核心因子，也是估值因子的近亲（高股息往往对应低估值）。它在 A 股长期有较稳定的正向超额收益，且在熊市中防御性突出。计算时把除息日落在过去 12 个月内的每股现金分红加总，再除以当日股价。方向：越高越好。

注意"高股息陷阱"：股息率 = 分红 / 股价，股价暴跌也会被动推高股息率。所以高股息要配合盈利稳定性、低杠杆一起看，避免买到"基本面恶化导致的伪高息"。

### 2. 分红率（派息率 / Payout Ratio）

定义：分红率 = 现金分红总额 / 归母净利润。

分红率反映的是**派息意愿**而非派息收益率。它的特殊之处在于：**不是越高越好**。分红率过高（接近或超过 100%）可能不可持续，甚至是借钱分红；分红率低也不一定差——高成长公司把利润留存再投资是合理的。所以分红率单独作因子效果一般，更适合作为辅助判据：在成熟、低增长的公司里，稳定适中的分红率是治理良好的信号。

### 3. 分红连续性 / 稳定性

定义：连续现金分红的年数；或过去 N 年分红的稳定程度（每股分红的波动率）。

"连续 10 年分红"传递的信息量很大——它意味着公司有持续的真实现金流、有稳定的治理、不靠融资续命。分红连续性是质量与治理的综合代理变量，在 A 股是较稳健的正向因子。方向：连续年数越长、分红越稳定越好。

### 4. 股票回购

定义：回购金额 / 总市值。

回购是与分红并列的股东回报方式——注销式回购等价于变相分红且更节税。但 A 股的回购数据较零散（多来自公告），整体规模也远小于美股，且部分回购用于股权激励而非注销，含义不同。可作为加分项纳入，但数据可得性和质量是现实约束，不必强求。

### 5. 综合股东回报率（Shareholder Yield）

定义：(现金分红 + 回购金额) / 总市值；更广义还可加上净偿还债务。

这是把分红和回购合并的"广义收益率"，回答的是"公司一年通过各种方式还给股东多少现金"。在回购更常见的美股它比单纯股息率更全面；A 股因回购规模有限，目前与股息率结果接近。方向：越高越好。

### 6. 股权稀释（负向）

定义：总股本同比增长率。

如果说分红回购是把现金还给股东，增发和送转就是反向稀释股东权益。频繁增发摊薄每股价值，是负向的"股东回报"。用 `valuation.capitalization` 的同比变化衡量。方向：股本增长越低越好（剔除送转影响后更准）。

---

## 四、标准变换

股东回报因子里，分红率和股权稀释方向需注意——股权稀释是负向因子，合成前取负。其余变换：股息率可做行业中性化与历史分位；分红连续性本身就是一个跨期统计量。这一类与估值因子相关性高，组合时要注意不要重复暴露同一风格；与盈利稳定性、低杠杆组合则互补性好。完整变换框架见《估值因子研究清单》第四节。

## 五、可直接运行的批量获取代码

```python
from jqfactor import winsorize_med, standardlize, neutralize
import pandas as pd
import numpy as np
import datetime

def get_shareholder_factors(date, stock_list):
    """取指定日期的股东回报因子（以股息率为主），已做标准预处理。
    注意：finance.STK_XR_XD 的字段名/单位请以聚宽 API 文档为准，
    A 股每股派息常按每 10 股登记，需相应换算。"""
    date = pd.Timestamp(date)
    start = date - pd.DateOffset(years=1)

    # 1. 查询过去 12 个月的分红记录
    div = finance.run_query(
        query(finance.STK_XR_XD).filter(
            finance.STK_XR_XD.code.in_(stock_list),
            finance.STK_XR_XD.a_xr_date >= start.date(),
            finance.STK_XR_XD.a_xr_date <= date.date(),
        ))
    # bonus_amount_rmb 处的字段名以文档为准；下面假设其为"每股现金红利(元)"
    div_per_share = div.groupby('code')['bonus_amount_rmb'].sum()

    # 2. 取当日股价，算股息率
    px = get_price(list(div_per_share.index), end_date=date,
                   count=1, fields=['close'], panel=False)
    close = px.set_index('code')['close']
    dividend_yield = (div_per_share / close).reindex(stock_list)

    # 3. 组装并标准预处理
    f = pd.DataFrame(index=stock_list)
    f['dividend_yield'] = dividend_yield.fillna(0)   # 无分红记 0
    for col in f.columns:
        s = f[col].replace([np.inf, -np.inf], np.nan)
        s = winsorize_med(s, scale=3, inclusive=True)
        s = standardlize(s, inf2nan=True)
        s = neutralize(s, how=['jq_l1', 'market_cap'], date=date)
        f[col] = s
    return f
```

分红率、分红连续性需要把分红记录与利润、与多年历史对齐，逻辑直接但代码更长，可在此基础上扩展。回购数据建议先确认聚宽是否提供对应接口再决定是否纳入。

## 六、因子方向与检验提示

股息率因子在 A 股近年有效性较强，且与市场风格相关——在避险情绪重、利率下行期表现尤其好。检验要点：股息率与估值因子（尤其 EP）相关性高，多因子模型里两者并用要做正交化，否则是重复暴露。分红率单独检验 IC 通常偏弱，更适合作为分层条件而非打分主因子。分红连续性这类跨期因子样本外稳定性较好，值得重点测。整体上股东回报因子换手低、防御性强，适合作为价值组合的稳定器。

---

## 七、注意事项与坑

- **高股息陷阱**：股价暴跌会被动抬高股息率，需结合盈利稳定性、低杠杆排除"伪高息"。
- **分红率非单调**：分红率不是越高越好——过高不可持续，过低未必差，慎将其作为打分主因子。
- **分红时点**：用除息日 / 实施日判断"是否已知"，不要用董事会预案日，否则引入未来函数。
- **送转干扰**：送股、转增不是现金回报，计算每股分红和股本增长率时要剔除。
- **字段单位**：分红表每股派息常按每 10 股登记；股本字段单位是万股——换算务必小心。
- **回购数据有限**：A 股回购数据零散、规模小且用途不一，作加分项即可，不必强行纳入。
